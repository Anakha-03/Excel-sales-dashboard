# 📊 Excel Sales Dashboard

## Overview

The **Excel Sales Dashboard** is an interactive dashboard developed using **Microsoft Excel**. It provides a comprehensive analysis of sales, profit, discounts, customers, products, and regional performance through Pivot Tables, Pivot Charts, Slicers, and dashboard visualizations.

The dashboard enables users to explore sales performance across different dimensions and supports data-driven business decision-making.

## Project Objectives

- Monitor sales and profit performance.
- Analyze customer and product sales.
- Evaluate sales across countries and market segments.
- Compare sales by discount bands.
- Track sales trends over time.
- Build an interactive Excel dashboard for business reporting.

---

## Dataset Information

The dashboard is built using sales transaction data containing:

- Customer Information
- Product Information
- Sales
- Gross Sales
- Discounts
- Profit
- Cost of Goods Sold (COGS)
- Units Sold
- Country
- Segment
- Date (Month, Quarter, Year)

## How to Use

1. Download the repository.
2. Open `Sales_Dashboard.xlsx` using Microsoft Excel.
3. Navigate to the **Dashboard** worksheet.
4. Use the slicers and filters to interact with the dashboard.
